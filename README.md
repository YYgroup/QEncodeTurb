This repository contains the source code for the arXiv paper: “Geometric quantum encoding of a turbulent field” (arXiv:2508.05346).

## File Overview
- `GeoEncodeTurb.ipynb`: This is the main program. It contains the core implementation and all post-processing steps.
- Other files: All other scripts in this repository are exclusively used for generating the plots and figures presented in the paper.

## Usage
Simply open and run `GeoEncodeTurb.ipynb` in Jupyter Notebook/Lab to execute the main program.
